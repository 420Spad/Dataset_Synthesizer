Installation
  1. Ensure your GPU drivers meeting the following criteria:

    -support at least OpenGL 4.3 -support at least Cuda 9.1.85

    Known good driver versions: 387.34, 390.67

  2. Install git LFS (large file storage): https://git-lfs.github.com/

    WARNING: If you pulled source before installing LFS, please remove the repo and sync again.

  3. Download the UE4 base code from https://gitlab-master.nvidia.com/NVStudios_DLToolkit/UE4_Base
    • This represents the base engine code with minimal changes to support various DLToolkit projects / plugins.
  4. chmod -R 777 in the downloaded project directory .

  5. Run the following:

    • $ ./Setup.sh
    • $ ./GenerateProjectFiles.sh
    • $ make

    The instructions above were derived from: https://wiki.unrealengine.com/Building_On_Linux#First_Time_Setup

    • Skip steps involving cloning EpicGames' github as you'll be pulling from the repository above.
    • Note the prerequisites and follow the "First Time Setup" section, detailing use of Setup.sh:
  6. Continue onto Usage
