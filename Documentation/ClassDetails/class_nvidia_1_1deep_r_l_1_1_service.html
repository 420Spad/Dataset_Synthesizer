Detailed Description

Service class includes the following AI Instance (agent) creation and Destruction. AI Instance Enumeration. Getter for Experiment interface. Getter for training state. Reader from config file. Tick which application must call each time period for this service. Limitation. A Service supports only one game training at same time. if you need to run multiple game training at same time, Multiple services need to be created. Service takes only one game state can include multiple AI instance state at same time. if you want to use totally separate game state, Please create multiple services.
